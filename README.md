# 職涯羅盤 Career Compass

> 你適合哪一種公司？薪資差異的背後，本質上是你在拿什麼，換取你的「人生選擇權」。

一個互動式、視覺化的職涯定位網站。改編自 林招煌 Alex《你適合哪種公司？薪資差異背後，真相是「人生選擇權」！》，並整合約 90+ 個公開研究來源（104、levels.fyi、主計總處、商周、HBR、80,000 Hours 等），把台灣職場最常見的四類公司拆解成可互動探索的內容。

純 **HTML / CSS / JavaScript**，無框架、無建構步驟、零外部依賴（圖表為自製 SVG），可直接以 `file://` 開啟，也可經本地伺服器瀏覽。

## ✨ 功能特色

- **2×2 定位模型** — 用「本土 ↔ 國際化」與「扁平彈性 ↔ 制度厚重」兩軸，把四類公司切成四象限，各對應一種核心修煉。
- **13 題情境測驗** — 12 題情境選擇 + 1 題職涯階段，依工作風格與偏好替你在兩軸計分。
- **三種結果視覺一次呈現**：
  - **主推薦** — 此刻最適合你的公司型態與「如何奪回選擇權」的修煉建議。
  - **2D 定位地圖**（自製 SVG）— 你的座標點動畫落在 2×2 象限上。
  - **契合度雷達圖**（自製 SVG）— 你對四類公司的傾向強弱。
  - **職涯階段導航** — 結合你的年齡階段給出修煉路線，並比對是否與該階段典型路線吻合。
- **深度內容** — 四類公司的薪資範圍、組織權責、領導遴選、文化與真實案例；35–40 歲中年危機數據；「平台能力 vs 個人能力」自我檢測表。
- **現代專業感設計** — 「墨色羅盤」編輯雜誌風：深墨藍底、羅盤金強調色、襯線標題、scroll-reveal 微互動，四類公司各有識別色，並支援 RWD（桌機 / 手機）。

## 🧭 核心模型

四類公司用兩個座標軸乾淨地切開：

```
                  制度厚重 / 防錯合規
                         │
        上市櫃公司         │        傳統外商
      (本土・科層)         │      (國際・矩陣)
   本土 ───────────────────┼─────────────────── 國際 / 跨國
        台灣中小企業        │         純美商
      (本土・草莽)         │      (國際・數字扁平)
                         │
                  扁平彈性 / 數字導向
```

| 公司型態 | 象限 | 核心修煉 | 識別色 |
|----------|------|----------|--------|
| 台灣中小企業 | 本土・草莽 | 生存力（從 0 到 1 破局） | 土橙 `#C8643C` |
| 上市櫃公司 | 本土・科層 | 組織政治力（協調、借力使力） | 鋼藍 `#4A7BA6` |
| 傳統外商 | 國際・矩陣 | 方法論（SOP、個人品牌） | 墨綠 `#2E8B7A` |
| 純美商 | 國際・數字扁平 | 市場變現力（數據說話） | 酒紅 `#B23A48` |

## 🔢 計分模型如何運作

1. 12 道情境題，每個選項帶有一組 `(dx, dy)` 向量：
   - `dx`：本土（−）↔ 國際化（+）
   - `dy`：扁平彈性（−）↔ 制度厚重（+）
2. 累加所有作答的向量，並依各題在兩軸上可達到的最大值**正規化到 `[-1, 1]`**，得到你的座標 `(x, y)`。
3. 計算你的座標與四個象限中心的歐氏距離，距離越近 → **契合度百分比越高**（雷達圖與排行用此值）。
4. 契合度最高者即為**主推薦**（平手時以選項傾向計次決勝）。
5. 第 13 題的職涯階段**不影響座標**，僅用於導航建議與「是否與此階段典型路線吻合」的比對。

計分相關常數與題庫全集中在 `js/data.js`，是整站的單一資料來源——調整題目、權重或公司資料只需改這一個檔案。

## 🛠 技術細節

- **無框架**：原生 HTML / CSS / JS，`<script>` 以 `window.CC` 命名空間共用資料，`file://` 與伺服器皆可運作。
- **自製 SVG 圖表**：雷達圖與 2D 定位地圖皆以 JS 動態產生 SVG，無 Chart.js 等外部依賴、離線可用。
- **狀態傳遞**：測驗結果同時寫入 `localStorage` 與 URL query string，確保 `file://` 下也能正確帶入結果頁。
- **字型**：Noto Serif TC（標題）+ Noto Sans TC（內文）+ Fraunces（拉丁字 / 數字），經 Google Fonts CDN 載入，並有系統字型 fallback。
- **RWD**：以 CSS Grid 為主，於 920px / 560px 斷點調整版面。

### 驗收
開發時以 Playwright（透過 uv）做自動化走查：載入全部 6 頁、自動跑完 13 題測驗→結果頁、檢查雷達圖 / 地圖 / 排行渲染與 console error。
