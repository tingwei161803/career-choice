> 狀態：已核准，進入實作

## 1. 目標

把原文「台灣職場四類公司」的剖析，做成一個**視覺化、互動式的純靜態網站**，幫使用者：

1. 理解四類公司的底層 DNA（組織權責、領導統御、主管遴選、文化）。
2. 透過互動測驗，找出此刻最適合自己的公司型態。
3. 結合職涯階段，得到具體的「修煉路線」導航建議。

設計理念引用原文金句：**「職涯不是選擇最厲害的公司，而是選擇最適合現在自己的舞台。」**

## 2. 核心模型：2×2 定位

四類公司用兩個座標軸切開，乾淨對應四象限：

- **X 軸**：本土 ←→ 國際化
- **Y 軸**：制度厚重（防錯/合規）←→ 扁平彈性（數字導向）

```
                  制度厚重 / 防錯合規
                         │
        上市櫃公司         │        傳統外商
      (本土・科層)         │      (國際・矩陣)
   本土 ───────────────────┼─────────────────── 國際/跨國
        台灣中小企業        │         純美商
      (本土・草莽)         │      (國際・數字扁平)
                         │
                  扁平彈性 / 數字導向
```

每類公司對應一種「核心修煉」：

| 公司型態 | 象限 | 核心修煉 | 識別色（暫定） |
|----------|------|----------|----------------|
| 台灣中小企業 | 本土・扁平 | 野蠻生存力（從 0 到 1 破局） | 土橙 |
| 上市櫃公司 | 本土・制度 | 組織政治力（協調、借力使力） | 沉藍 |
| 傳統外商 | 國際・制度 | 方法論 / 國際視野（SOP、個人品牌） | 墨綠 |
| 純美商 | 國際・扁平 | 市場價值變現力（數據說話） | 酒紅 |

## 3. 測驗機制（三種結果呈現全做）

- **題數**：約 12 題情境選擇題 + 1 題職涯階段（年齡/年資）。
- **計分**：每題選項對兩軸（X：本土↔國際、Y：制度↔扁平）加權累分，並換算成對四類公司的「契合度百分比」（依使用者座標與各象限中心的距離）。
- **結果頁同時呈現**：
  1. **主推薦（單軸）**：最契合的公司型態 + 該型態的核心修煉建議（取自原文「如何奪回選擇權」）。
  2. **2D 定位地圖（雙軸）**：使用者座標點以動畫落在 2×2 象限圖上。
  3. **契合度雷達圖**：對四類公司的傾向強弱，自製 SVG，零外部依賴、離線可看。
  4. **職涯階段導航**：結合年齡階段（25-30 初期 / 30-40 黃金期）給出路線建議與「平台 ≠ 能力」提醒。

## 4. 頁面結構（多 HTML，index 進入）

| 檔案 | 內容 |
|------|------|
| `index.html` | Hero 首頁：主題、Alex 金句、四類公司預覽卡、開始測驗 CTA |
| `companies.html` | 四類公司深度剖析：四維度 + 研究擴充的薪資範圍、真實案例、優缺點 |
| `quiz.html` | 12+1 題互動測驗，進度條、流暢轉場 |
| `result.html` | 動態結果頁（資料經 localStorage / URL 參數傳入）：主推薦 + 2D 地圖 + 雷達圖 + 導航 |
| `navigator.html` | 職涯導航：年齡階段時間軸、決策樹、中年危機提醒 |
| `sources.html` | 資料來源與方法說明（研究引用，建立可信度） |

**共用資產**：
- `css/styles.css`：設計系統（色彩、字型、間距、元件）。
- `js/data.js`：公司資料 + 題庫 + 計分模型常數（單一資料來源）。
- `js/quiz.js`：測驗流程與計分。
- `js/result.js`：結果計算、SVG 雷達圖與 2D 地圖繪製。

## 5. 技術與工具

- 純 HTML/CSS/JS，**無框架**。可直接 `file://` 打開或經本地伺服器瀏覽。
- 圖表（雷達圖、2D 地圖）**自製 SVG**，零外部依賴、離線可用。
- 中文字型 Noto Sans TC（Google Fonts CDN）+ 系統字型 fallback。
- **uv** 用於：跑本地伺服器（`uv run python -m http.server`）與研究/資料處理腳本。

## 6. 視覺風格（現代專業感）

- 沉穩主視覺 + 一個強調色（商務藍/墨綠系）；四類公司各有識別色。
- 清晰排版、充足留白、卡片式資訊、微互動（hover、scroll 進場），不浮誇。
- 響應式：桌機與手機皆可用。
